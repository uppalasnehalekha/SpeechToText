Run everything in Python 2.7(3.6 is not compatible because of JSON Serializable Error).
Django should also work on Python 2.7
requests python library should be installed.
gensim package should also be installed
googletrans package should also be installed

Open SpeechToText folder and open cmd in this folder and enter "python manage.py runserver" command.
Click on file button and select any language file from SpeechToText folder.
Select the language and click on convert.
Wait for 2-3 minutes as it takes some time for watson to give the response.
Download the transcripts for later use.
